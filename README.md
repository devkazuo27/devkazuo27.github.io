# Portafolio — Naren Sánchez

Portafolio personal: bots, automatizaciones, herramientas a medida y sitios web.

- **Stack**: HTML + CSS + JavaScript vanilla, sin dependencias ni build.
- **Hosting**: GitHub Pages.

## Desarrollo local

Abre `index.html` en el navegador, o sirve la carpeta:

```bash
npx serve .
```

## Estructura

```
index.html      # Página única (hero, proyectos, servicios, stack, contacto)
css/styles.css  # Sistema de diseño (tokens CSS) + estilos
js/main.js      # Menú móvil, reveal on scroll, año dinámico
```
